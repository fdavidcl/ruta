ruta.layerOutputs {ruta}R Documentation

Get outputs from any layer from a trained model and new data.

Description

Get outputs from any layer from a trained model and new data.

Usage

ruta.layerOutputs(model, task, layerInput = 1, layerOutput, ...)

Arguments

model

A "rutaModel" object.

task

A "rutaTask" object.

layerInput

An integer indicating the index of the layer in which data will be injected.

layerOutput

An integer indicating the index of the layer to be obtained.

...

Custom parameters for internal prediction function.

Value

A matrix containing layer outputs for each instance in the given task.
